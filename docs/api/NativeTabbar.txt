<p>The NativeTabbar methods let you customize the Rhodes tabbar at runtime.</p>
<p>No more than 4 buttons can be created in Windows Phone 8 toolbar. A toolbar separator doesn&rsquo;t exist in Windows Phone 8. On Windows Mobile/CE there are no visible tabs.</p>

<h2>Enabling the API</h2>

<p>This API is part of the <code>coreapi</code> extension that is included automatically.</p>

<pre><code>:::ruby
extensions: ["coreapi"]
</code></pre>
