<p>The NativeToolbar methods let you create and remove toolbar at runtime.</p>
<p> At one time only Toolbar or Tabbar can present in application &ndash; Toolbar and Tabbar can not be used both in one time.</p>

<h2>Enabling the API</h2>

<p>This API is part of the <code>coreapi</code> extension that is included automatically.</p>

<pre><code>:::ruby
extensions: ["coreapi"]
</code></pre>
