# Error codes:
# ERR_NONE = 0
# ERR_NETWORK = 1
# ERR_REMOTESERVER = 2
# ERR_RUNTIME = 3
# ERR_UNEXPECTEDSERVERRESPONSE = 4
# ERR_DIFFDOMAINSINSYNCSRC = 5
# ERR_NOSERVERRESPONSE = 6
# ERR_CLIENTISNOTLOGGEDIN = 7
# ERR_CUSTOMSYNCSERVER = 8
# ERR_UNATHORIZED = 9

def login_callback
  error_code = @params['error_code'].to_i
  if error_code == Rho::RhoError::ERR_NONE
    # run sync if we were successful
    WebView.navigate Rho::RhoConfig.options_path
    RhoConnectClient.doSync
  else
    if error_code == Rho::RhoError::ERR_CUSTOMSYNCSERVER
      @msg = @params['error_message']
    end
  
    if not @msg or @msg.length == 0
      @msg = Rho::RhoError.new(error_code).message
    end
  
    WebView.navigate(
      url_for(:action =&gt; :login, :query =&gt; {:msg =&gt; @msg})
    )
  end
end

                                def show_page
                                    @contacts = Contact.find(
                                        :all,
                                        :conditions =&gt; {
                                        {
                                            :func =&gt; 'LOWER',
                                            :name =&gt; 'FirstName',
                                            :op =&gt; 'LIKE'
                                        } =&gt; @params[:FirstName],
                                        {
                                            :func =&gt; 'LOWER',
                                            :name=&gt;'LastName',
                                            :op=&gt;'LIKE'
                                        } =&gt; @params[:LastName],
                                        {
                                            :func=&gt;'LOWER',
                                            :name=&gt;'Company',
                                            :op=&gt;'LIKE'
                                        } =&gt; @params[:Company],
                                        },
                                        :op =&gt; 'OR',
                                        :select =&gt; ['FirstName','LastName', 'Company'],
                                        :per_page =&gt; page_size,
                                        :offset =&gt; page * page_size

<a name='Remarks'></a>
<h2><i class='icon-warning-sign'></i>Remarks</h2>

<a name='r0'></a><div class='accordion property' id='r0'><div class="accordion-group"><div class="accordion-heading"><span class="accordion-toggle" data-toggle="collapse"  href="#cRemark0"><strong>Notification Parameters</strong><i class="icon-chevron-down pull-left"></i></span></div><div id="cRemark0" class="accordion-body collapse in">  <div class="accordion-inner"><p>When the notification is called, it will receive a variable called @params, just like a normal Rhodes controller action.</p>

<h3>Common Parameters</h3>

<p>These parameters are included in all notifications.</p>

<ul>
<li>@params[&ldquo;source_id&rdquo;] &ndash; The id of the current model that is synchronizing.</li>
<li>@params[&ldquo;source_name&rdquo;] &ndash; Name of the model (i.e. &ldquo;Product&rdquo;)</li>
<li>@params[&ldquo;sync_type&rdquo;] &ndash; Type of sync used for this model: &ldquo;incremental&rdquo; or &ldquo;bulk&rdquo;</li>
<li>@params[&ldquo;status&rdquo;] &ndash; Status of the current sync process: &ldquo;in_progress&rdquo;, &ldquo;error&rdquo;, &ldquo;ok&rdquo;, &ldquo;complete&rdquo;, &ldquo;schema-changed&rdquo;</li>
</ul>


<h3>&ldquo;in_progress&rdquo; &ndash; incremental sync</h3>

<ul>
<li>@params[&ldquo;total_count&rdquo;] &ndash; Total number of records that exist for this RhoConnect source.</li>
<li>@params[&ldquo;processed_count&rdquo;] &ndash; Number of records included in the sync page.</li>
<li>@params[&ldquo;cumulative_count&rdquo;] &ndash; Number of records the SyncEngine has processed so far for this source.</li>
</ul>


<h3>&ldquo;in_progress&rdquo; &ndash; bulk sync</h3>

<ul>
<li><p>@params[&ldquo;bulk_status&rdquo;] &ndash; The state of the bulk sync process:</p>

<ul>
<li>&ldquo;start&rdquo;: when bulk sync start and when specific partition is start syncing</li>
<li>&ldquo;download&rdquo;: when client start downloading database from server</li>
<li>&ldquo;change_db&rdquo;: when client start applying new database</li>
<li>&ldquo;blobs&rdquo;: when client start downloading remote blob files</li>
<li>&ldquo;ok&rdquo;: when sync of partition finished without error</li>
<li>&ldquo;complete&rdquo;: when bulk sync finished for all partitions without errors</li>
</ul>
</li>
<li><p>@params[&ldquo;partition&rdquo;] &ndash; Current bulk sync partition.</p></li>
</ul>


<h3>&ldquo;error&rdquo;</h3>

<ul>
<li>@params[&ldquo;error_code&rdquo;] &ndash; HTTP response code of the RhoConnect server error: 401, 500, 404, etc.</li>
<li>@params[&ldquo;error_message&rdquo;] &ndash; Response body (if any)</li>
<li>@params[&ldquo;server_errors&rdquo;] &ndash; Hash of Type objects of RhoConnect adapter error (if exists): &ldquo;login-error&rdquo;, &ldquo;query-error&rdquo;, &ldquo;create-error&rdquo;, &ldquo;update-error&rdquo;, &ldquo;delete-error&rdquo;, &ldquo;logoff-error&rdquo;</li>
</ul>


<p>For &ldquo;login-error&rdquo;, &ldquo;query-error&rdquo;, &ldquo;logoff-error&rdquo;: Type object is hash contains &lsquo;message&rsquo; from server: @params[&ldquo;server_errors&rdquo;][&ldquo;query-error&rdquo;][&lsquo;message&rsquo;]</p>

<p>For &ldquo;create-error&rdquo;, &ldquo;update-error&rdquo;, &ldquo;delete-error&rdquo;: Type object is hash each containing an &ldquo;object&rdquo; as a key (that failed to create) and a corresponding &ldquo;message&rdquo; and &ldquo;attributes&rdquo;: @params[&ldquo;server_errors&rdquo;][&ldquo;create-error&rdquo;][object][&lsquo;message&rsquo;], @params[&ldquo;server_errors&rdquo;][&ldquo;create-error&rdquo;][object][&lsquo;attributes&rsquo;]</p>

<p>&ldquo;create-error&rdquo; has to be handled in sync callback. Otherwise sync will stop on this model. To fix create errors you should call Model.on_sync_create_error or SyncEngine.on_sync_create_error
&ldquo;ok&rdquo;
* @params[&ldquo;total_count&rdquo;] &ndash; Total number of records that exist for this RhoConnect source.
* @params[&ldquo;processed_count&rdquo;] &ndash; Number of records included in the last sync page.
* @params[&ldquo;cumulative_count&rdquo;] &ndash; Number of records the SyncEngine has processed so far for this source.</p>

<h3>&ldquo;complete&rdquo;</h3>

<p>This status returns only when the SyncEngine process is complete.</p>

<h3>&ldquo;schema-changed&rdquo;</h3>

<p>This status returns for bulk-sync models that use FixedSchema when the schema has changed in the RhoConnect server. In this scenario the sync callback should notify the user with a wait screen and start the bulk sync process.</p>
