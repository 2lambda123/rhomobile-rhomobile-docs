<p>The MediaPlayer extension is used to playback different types of audio file. It also allows the playback of Ringtones for the respective platform.</p>

<h2>Enabling the API</h2>

<p>In order to use this API you must include the following extension in your <code>build.yml</code></p>

<pre><code>:::ruby
extensions: ["mediaplayer"]
</code></pre>
