<a name='e0'></a><div class='accordion property' id='e0'><div class="accordion-group"><div class="accordion-heading"><span class="accordion-toggle" data-toggle="collapse"  href="#cExample0"><strong>Execute javascript</strong><i class="icon-chevron-down pull-left"></i></span></div><div id="cExample0" class="accordion-body collapse in">  <div class="accordion-inner">

                   You can call the WebView and make it execute Javascript code from a Ruby controller. This can be particularly helpful in order to reuse Javascript functionality 
                   that is already included in your pages 
               <ul class='nav nav-tabs' id='exI0-S0Tab'><li class='active'><a href='#exI0-S0RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI0-S0RUBY'><pre class='CodeRay'><code>:::ruby
           
    #call javascript alert on the current page
    Rho::WebView.executeJavascript("alert('This is Webview.executejavascript function');")

                   

               </code></pre></div></div>

                    As an example, you can invoke JQuery Mobile's changePage
               <ul class='nav nav-tabs' id='exI0-S1Tab'><li class='active'><a href='#exI0-S1RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI0-S1RUBY'><pre class='CodeRay'><code>:::ruby
           
    #call JQuery Mobile from a Ruby controller
    Rho::WebView.executeJavascript("$.mobile.changePage('#my_page');")
                   

               </code></pre></div></div>

                    You can also call your own functions                    
               <ul class='nav nav-tabs' id='exI0-S2Tab'><li class='active'><a href='#exI0-S2JS' data-toggle='tab'>Javascript</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI0-S2JS'><pre class='CodeRay'><code>:::javascript
           
           function greet(name) {
                alert("Hello, "+name);  
           }
                   

               </code></pre></div></div>

                    From the Ruby controller we can execute the Javascript function that may be in the view.
                <ul class='nav nav-tabs' id='exI0-S3Tab'><li class='active'><a href='#exI0-S3RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI0-S3RUBY'><pre class='CodeRay'><code>:::ruby
           
    #call a custom Javascript function from Ruby
    Rho::WebView.executeJavascript("greet('John Doe');")
                   

               </code></pre></div></div>  </div></div></div></div><a name='e1'></a><div class='accordion property' id='e1'><div class="accordion-group"><div class="accordion-heading"><span class="accordion-toggle" data-toggle="collapse"  href="#cExample1"><strong>Use full-screen mode</strong><i class="icon-chevron-down pull-left"></i></span></div><div id="cExample1" class="accordion-body collapse in">  <div class="accordion-inner">

                   The WebView can be set to use all the available screen real-estate by turning on "full-screen" mode 
               <ul class='nav nav-tabs' id='exI1-S0Tab'><li class='active'><a href='#exI1-S0RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI1-S0RUBY'><pre class='CodeRay'><code>:::ruby
           
           # Toggle fullScreen on / off
        Rho::WebView.fullScreen = !Rho::WebView.fullScreen 
                   

               </code></pre></div></div>

                   This property can also be assigned to in Javascript   
               <ul class='nav nav-tabs' id='exI1-S1Tab'><li class='active'><a href='#exI1-S1JS' data-toggle='tab'>Javascript</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI1-S1JS'><pre class='CodeRay'><code>:::javascript
           
        // Fullscreen can also be invoked from Javascript
        Rho.WebView.fullScreen = true;
        
        Rho.WebView.fullScreen = false;
                   

               </code></pre></div></div>  </div></div></div></div><a name='e2'></a><div class='accordion property' id='e2'><div class="accordion-group"><div class="accordion-heading"><span class="accordion-toggle" data-toggle="collapse"  href="#cExample2"><strong>Navigate</strong><i class="icon-chevron-down pull-left"></i></span></div><div id="cExample2" class="accordion-body collapse in">  <div class="accordion-inner">

               </code></pre></div></div>

                   You can also navigate outside of your application, to an external site
               <ul class='nav nav-tabs' id='exI2-S1Tab'><li class='active'><a href='#exI2-S1RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI2-S1RUBY'><pre class='CodeRay'><code>:::ruby

               </code></pre></div></div>

      Rho.WebView.navigateBack();
    
                   

               </code></pre></div></div>  </div></div></div></div><a name='e4'></a><div class='accordion property' id='e4'><div class="accordion-group"><div class="accordion-heading"><span class="accordion-toggle" data-toggle="collapse"  href="#cExample4"><strong>Refresh the page</strong><i class="icon-chevron-down pull-left"></i></span></div><div id="cExample4" class="accordion-body collapse in">  <div class="accordion-inner">

                   Reload the current URL into the WebView. This is useful especially after having updated data that must be now shown to the user 
               <ul class='nav nav-tabs' id='exI4-S0Tab'><li class='active'><a href='#exI4-S0RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI4-S0RUBY'><pre class='CodeRay'><code>:::ruby
           
    # Update database or variables here
    # ...      
    # Refresh the current page to show new information
    Rho::WebView.refresh
                   

               </code></pre></div></div>

                   By default, "refresh" will update the current view. If you are using the native Tabbar and have multiple WebViews, you can specify which one to refresh
               <ul class='nav nav-tabs' id='exI4-S1Tab'><li class='active'><a href='#exI4-S1RUBY' data-toggle='tab'>Ruby</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI4-S1RUBY'><pre class='CodeRay'><code>:::ruby
           
     # Reload the page on WebView number 2
      Rho::WebView.refresh(2)
                   

               </code></pre></div></div>

                   Also available from Javascript
               <ul class='nav nav-tabs' id='exI4-S2Tab'><li class='active'><a href='#exI4-S2JS' data-toggle='tab'>Javascript</a></li></ul><div class='tab-content'><div class='tab-pane active' id='exI4-S2JS'><pre class='CodeRay'><code>:::javascript
           
     // Reload the current page
      Rho.WebView.refresh()

     // Reload the page on WebView number 3
      Rho.WebView.refresh(3)
      
                   

               </code></pre></div></div>  </div></div></div></div>
