<p>The Mobile Payment module implements the functions required to work with the D180 Bluetooth Payment MPOS.</p>
<p>Before an application can use this module, the device must be paired via Bluetooth the mobile device requesting use.</p>

<pre><code>If this is not done, no functions will work and the Open function will return an error of not paired.

## Enabling the API
In order to use this API you must include the following extension in your `build.yml`.
:::ruby
extensions: ["mobile_payment"]

The `mobile_payment` extension is also included automatically if you specify the following in your `build.yml`.
:::ruby
extensions: ["rhoelements"]
