#NativeMenubar


## Overview
<p>The NativeMenubar API lets you customize the Windows Mobile/CE native menu buttons.</p>
<h2>Enabling the API</h2>

<p>This API is part of the <code>coreapi</code> extension that is included automatically.</p>

<pre><code>:::ruby
extensions: ["coreapi"]
</code></pre>
